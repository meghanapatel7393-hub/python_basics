Lesson 21: Generators & Iterators

1. What is an Iterator?

An iterator is an object that allows you to traverse through all elements of a collection one by one.

numbers = [1, 2, 3]
it = iter(numbers)
print(next(it))
print(next(it))
print(next(it))
Iterators use __iter__() and __next__() methods internally.

2. Custom Iterator Example

class CountUp:
  def __init__(self, max):
    self.max = max
    self.current = 1

  def __iter__(self):
    return self

  def __next__(self):
    if self.current > self.max:
      raise StopIteration
    val = self.current
    self.current += 1
    return val

3. What is a Generator?

A generator is a simpler way to create iterators using the yield keyword.

Generator remembers its state between function calls.

4. Generator Function Example

def count_up(max):
  num = 1
  while num <= max:
    yield num
    num += 1

for n in count_up(5):
  print(n)

5. Generator Expression

gen = (x * x for x in range(5))
print(next(gen))
print(next(gen))

6. Generator vs Iterator

Iterator Generator
More code Less code
Uses __iter__ & __next__ Uses yield
Complex Simple & clean
Manual StopIteration Automatic StopIteration

7. Memory Efficiency

Generators generate values one at a time, saving memory for large datasets.

nums = (i for i in range(1000000)) # memory efficient
