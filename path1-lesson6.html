Lesson 6: File Handling & Context Managers

1. What is File Handling?

File handling allows Python programs to read, write, update, and delete files.

Interview line:
“File handling is used to persist data permanently.”

2. Opening a File

file = open("data.txt", "r")
Mode Description
r Read (default)
w Write (overwrite)
a Append
r+ Read + Write

3. Reading from a File

file = open("data.txt", "r")
print(file.read())
file.close()
file = open("data.txt", "r")
print(file.readline())
file = open("data.txt", "r")
print(file.readlines())

4. Writing to a File

file = open("data.txt", "w")
file.write("Hello Python")
file.close()
Write mode overwrites existing content.

5. Appending to a File

file = open("data.txt", "a")
file.write("\nNew line added")
file.close()

6. Why Close a File?

  • Free system resources
  • Prevent data corruption
  • Ensure data is saved

7. Context Manager (with Statement)

The with statement automatically handles file closing.

with open("data.txt", "r") as file:
  print(file.read())
Best practice for file handling.

8. File Handling with Exception Safety

try:
  with open("data.txt", "r") as file:
    print(file.read())
except FileNotFoundError:
  print("File not found")

9. Interview Comparison

Approach Manual with Statement
