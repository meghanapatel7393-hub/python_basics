Lesson 7: OOP Basics – Class, Object & Constructor

1. What is OOP?

Object-Oriented Programming (OOP) is a programming paradigm that organizes code using classes and objects.

Interview line:
“OOP helps in code reusability, scalability, and maintainability.”

2. What is a Class?

A class is a blueprint or template for creating objects.

class Person:
  pass

3. What is an Object?

An object is an instance of a class.

p1 = Person()
p2 = Person()
Each object has its own memory.

4. Constructor (__init__)

A constructor is a special method that runs automatically when an object is created.

class Person:
  def __init__(self, name, age):
    self.name = name
    self.age = age
p1 = Person("Bhavesh", 22)
print(p1.name)
print(p1.age)

5. What is self?

self refers to the current object.

Interview line:
“self is a reference to the current instance of the class.”

6. Instance Variables & Methods

class Student:
  def __init__(self, name, marks):
    self.name = name
    self.marks = marks

  def display(self):
    print(self.name, self.marks)
s1 = Student("Amit", 85)
s1.display()

7. Class Variables

Class variables are shared among all objects.

class Company:
  company_name = "TechCorp"

  def __init__(self, emp):
    self.emp = emp
Instance variables belong to objects, class variables belong to class.
