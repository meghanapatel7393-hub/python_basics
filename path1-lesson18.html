1. Python Modules

A module is a Python file (.py) containing functions, classes, or variables that can be imported into other Python programs.

# example_module.py
def greet(name):
  print(f"Hello {name}")

# main.py
import example_module
example_module.greet("Bhavesh")
Use import module_name to include any module.

2. Import Variations

  • import module – Import full module
  • from module import func – Import specific function/class
  • import module as alias – Create alias for module
  • from module import * – Import everything (not recommended)
from math import sqrt
print(sqrt(16)) # Output: 4

3. Python Packages

A package is a directory containing a special __init__.py file and multiple modules.

# folder structure
mypackage/
  __init__.py
  module1.py
  module2.py

# usage
from mypackage import module1
module1.some_function()

4. Python Standard Library

  • Modules included with Python installation
  • Examples: os, sys, math, datetime, json, random
  • Reduces the need to write code from scratch

5. pip – Python Package Installer

pip is used to install third-party packages from PyPI (Python Package Index).

# Install a package
pip install requests

# Upgrade a package
pip install --upgrade requests

# Uninstall a package
pip uninstall requests
Use import package_name after installation.

6. sys & os Modules

import sys, os
print(sys.version) # Python version
print(os.getcwd()) # Current working directory

7. Best Practices

  • Use virtual environments (venv) to manage project dependencies
  • Use meaningful package and module names
  • Organize large projects into packages for clarity
  • Avoid from module import *

  3. Install a package using pip and use it in a script
