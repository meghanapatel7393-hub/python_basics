Lesson 13: Context Managers & Advanced File Handling

1. File Handling Basics

Python allows you to read and write files using open().

# Open a file in read mode
f = open("example.txt", "r")
content = f.read()
f.close()
Always close the file to free resources.

2. Context Managers

Context managers simplify file handling and automatically close the file.

with open("example.txt", "r") as f:
  content = f.read()
print(content)
The with statement ensures the file is closed even if an error occurs.

3. Writing to a File

with open("output.txt", "w") as f:
  f.write("Hello Bhavesh!")
Modes: r, w, a, rb, wb

4. Reading Lines

with open("example.txt", "r") as f:
  for line in f:
    print(line.strip())

5. File Handling Methods

Method Purpose
read() Read entire file
readline() Read one line
readlines() Return all lines as a list
write() Write to file
writelines() Write multiple lines

6. Custom Context Managers

class FileManager:
  def __init__(self, filename, mode):
    self.filename = filename
    self.mode = mode

  def __enter__(self):
    self.file = open(self.filename, self.mode)
    return self.file

  def __exit__(self, exc_type, exc_val, exc_tb):
    self.file.close()
with FileManager("output.txt", "w") as f:
  f.write("Hello from custom context manager!")
Implements __enter__ and __exit__ methods.

7. File Handling Tips

  • Always use context managers to prevent file leaks
  • Use with open() for safer code
  • Handle exceptions for production code
