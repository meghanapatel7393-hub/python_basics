Lesson 30: Python Performance Optimization & Profiling

1. Why Performance Matters?

  • Faster applications
  • Better user experience
  • Lower server costs
  • Scalability

2. Common Performance Problems

  • Nested loops
  • Unnecessary calculations
  • Inefficient data structures
  • Repeated I/O operations

3. Time Complexity Basics

Complexity Example
O(1) Access by index
O(n) Single loop
O(n²) Nested loops

4. Using Efficient Data Structures

Use Instead of
set list (for lookup)
dict multiple if-else

5. List Comprehension vs Loop

# Slower
result = []
for i in range(10):
  result.append(i*i)

# Faster
result = [i*i for i in range(10)]

6. Profiling with time Module

import time

start = time.time()
# code here
end = time.time()
print("Time:", end - start)

7. Profiling with cProfile

import cProfile

def test():
  for i in range(100000):
    i * i

cProfile.run("test()")
Shows function-wise execution time.

8. Avoid Premature Optimization

First write clean code, then optimize only bottlenecks.

Optimization without profiling can make code worse.

9. Caching with lru_cache

from functools import lru_cache

@lru_cache(maxsize=None)
def fib(n):
  if n < 2:
    return n
  return fib(n-1) + fib(n-2)

10. Performance Tips

  • Use generators for large data
  • Avoid global variables
  • Minimize disk & network I/O
  • Profile before optimizing
