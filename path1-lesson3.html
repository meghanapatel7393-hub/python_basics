Lesson 3: Lambda, map(), filter(), reduce()

1. Lambda Functions

A lambda function is a small anonymous function written in one line.

# Normal function
def square(x):
  return x * x

# Lambda function
square = lambda x: x * x
print(square(5))
Interview line:
‚ÄúLambda functions are anonymous, single-expression functions.‚Äù

2. Why Use Lambda?

  • Short and clean code
  • Used with map, filter, reduce
  • One-time use functions
Lambdas should be simple. Avoid complex logic.

3. map() Function

map() applies a function to every element of an iterable.

nums = [1, 2, 3, 4]

result = map(lambda x: x * 2, nums)
print(list(result))

Output:

[2, 4, 6, 8]
map() transforms data.

4. filter() Function

filter() returns elements that satisfy a condition.

nums = [1, 2, 3, 4, 5]

result = filter(lambda x: x % 2 == 0, nums)
print(list(result))

Output:

[2, 4]
filter() selects data.

5. reduce() Function

reduce() reduces an iterable to a single value.

from functools import reduce

nums = [1, 2, 3, 4]
result = reduce(lambda a, b: a + b, nums)
print(result)

Output:

10
reduce() accumulates values.

6. map vs filter vs reduce (Interview Comparison)

Function Purpose Output
map() Transform each element Iterable
filter() Select elements Iterable
reduce() Combine elements Single value

7. When NOT to Use Lambda?

  • Complex logic
  • Multiple statements
  • Reusable logic
