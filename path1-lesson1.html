1. Is Python Compiled or Interpreted?

Interview-correct answer:
Python is an interpreted language, but internally it is first compiled into bytecode.

Python source code (.py) is compiled into bytecode (.pyc) and then executed by the Python Virtual Machine (PVM).

2. Variables & Memory (Interview Favorite)

In Python, a variable is not a container. It is a reference to an object in memory.

a = 10
b = a

Both a and b point to the same object 10.

Interview Trap: Python caches small integers for performance.
a = 10
b = 10
print(a is b) # True

3. Mutable vs Immutable (VERY IMPORTANT)

Immutable Data Types

  • int
  • float
  • string
  • tuple
x = 10
x = x + 1

A new object is created in memory.

Mutable Data Types

  • list
  • dictionary
  • set
a = [1, 2]
b = a
b.append(3)
print(a)

Output: [1, 2, 3]

4. is vs ==

  • == ‚Üí compares values
  • is ‚Üí compares memory location
a = [1, 2]
b = [1, 2]
print(a == b) # True
print(a is b) # False
Interview line:
‚Äú== checks value equality, is checks object identity.‚Äù

5. Function Arguments (Tricky Interview Concept)

Python uses pass-by-object-reference.

def change(x):
  x.append(4)

lst = [1, 2, 3]
change(lst)
print(lst)

Output: [1, 2, 3, 4]

def change(x):
  x = x + 1

a = 10
change(a)
print(a)

Output: 10

6. Shallow Copy vs Deep Copy

import copy

a = [[1, 2], [3, 4]]
b = copy.copy(a)
c = copy.deepcopy(a)
  • Shallow copy: inner objects shared
  • Deep copy: fully independent objects
