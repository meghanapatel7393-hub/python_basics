Lesson 4: Flask Routing & URL Parameters

1. Flask Routing

Routing maps a URL path to a Python function.

from flask import Flask
app = Flask(__name__)

@app.route("/")
def home():
  return "Home Page"

@app.route("/about")
def about():
  return "About Page"
Visit / → Home Page, /about → About Page

2. URL Parameters (Dynamic Routing)

You can pass variables in the URL using <parameter>.

@app.route("/user/<username>")
def show_user(username):
  return f"Hello, {username}!"
Example: /user/Bhavesh → "Hello, Bhavesh!"

3. Multiple Parameters

@app.route("/post/<int:post_id>/<action>")
def post_action(post_id, action):
  return f"Post {post_id} will be {action}"
Visit /post/10/edit → "Post 10 will be edit"

4. URL Converter Types

Type Description
string Default, accepts text without slashes
int Accepts integers only
float Accepts floating point numbers
path Accepts text including slashes
uuid Accepts UUID strings

5. URL Building (Optional)

from flask import url_for

@app.route("/profile")
def profile():
  return "Profile Page"

# url_for("profile") → "/profile"
Helps generate URLs dynamically instead of hardcoding.
