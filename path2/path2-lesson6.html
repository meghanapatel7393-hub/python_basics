Lesson 6: Flask Forms & Handling User Input

1. What are Forms?

Forms allow users to send data to the server via HTTP POST or GET requests.

Common use: login, registration, feedback, contact forms.

2. HTML Form Example

<form action="/submit" method="POST">
  <input type="text" name="username" placeholder="Enter username">
  <input type="submit" value="Submit">
</form>

3. Flask Route to Handle Form

from flask import Flask, request, render_template
app = Flask(__name__)

@app.route("/submit", methods=["GET", "POST"])
def submit():
  if request.method == "POST":
    username = request.form["username"]
    return f"Hello, {username}!"
  return render_template("form.html")
request.form["username"] gets the value from the input field.

4. Handling GET Requests

@app.route("/search")
def search():
  query = request.args.get("q")
  return f"Searching for: {query}"

# Access via /search?q=python

5. Templates with Forms

<!-- form.html -->
<form action="/submit" method="POST">
  <input type="text" name="username" placeholder="Enter username">
  <input type="submit" value="Submit">
</form>
Rendering templates keeps HTML separate from Python logic.

6. Common Form Tips

  • Always validate input (required, length, type)
  • Use POST for sensitive data
  • Use GET for queries/searches
  • Escape HTML to prevent XSS attacks
