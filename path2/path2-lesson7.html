Lesson 7: Flask Templates with Loops & Conditionals

1. Looping in Templates

Use {% for %} to loop over a list or dictionary passed from Flask.

@app.route("/fruits")
def fruits():
  items = ["Apple", "Banana", "Orange"]
  return render_template("fruits.html", items=items)
Now we can use {% for %} in fruits.html

2. Loop Example in Template

<!-- fruits.html -->
<ul>
{% for fruit in items %}
  <li>{{ fruit }}</li>
{% endfor %}
</ul>
This renders a list of fruits dynamically.

3. Conditional Statements

Use {% if %} to conditionally display content.

@app.route("/check/<int:age>")
def check(age):
  return render_template("check.html", age=age)
Pass age to template and use {% if %}

4. Conditional Example in Template

<!-- check.html -->
{% if age >= 18 %}
  <p>You are an adult.</p>
{% else %}
  <p>You are a minor.</p>
{% endif %}

5. Combining Loops & Conditionals

<ul>
{% for fruit in items %}
  {% if fruit == "Banana" %}
    <li>{{ fruit }} (Yummy!)</li>
  {% else %}
    <li>{{ fruit }}</li>
  {% endif %}
{% endfor %}
</ul>
You can use loops with conditionals for dynamic, intelligent content rendering.
