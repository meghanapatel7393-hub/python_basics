Lesson 13: Flask File Uploads & Handling Media

1. Why File Uploads?

Many apps require users to upload files such as images, PDFs, or documents.

Flask provides simple ways to handle file uploads and save them on the server.

2. HTML Form for File Upload

<form method="POST" action="/upload" enctype="multipart/form-data">
  <input type="file" name="file">
  <input type="submit" value="Upload">
</form>
enctype="multipart/form-data" is required for file uploads.

3. Flask Route to Handle Upload

from flask import Flask, request, redirect, url_for
import os
app = Flask(__name__)

UPLOAD_FOLDER = "uploads"
app.config["UPLOAD_FOLDER"] = UPLOAD_FOLDER

@app.route("/upload", methods=["GET", "POST"])
def upload_file():
  if request.method == "POST":
    file = request.files["file"]
    if file.filename != "":
      file.save(os.path.join(app.config["UPLOAD_FOLDER"], file.filename))
      return "File uploaded successfully!"
  return ''' <form method="POST" enctype="multipart/form-data"> <input type="file" name="file"> <input type="submit"> </form> '''

4. Notes & Best Practices

  • Create the uploads folder before uploading files
  • Validate file extensions for security (jpg, png, pdf)
  • Use secure_filename from werkzeug.utils to prevent unsafe filenames
  • Limit file size using app.config["MAX_CONTENT_LENGTH"]

5. Using secure_filename

from werkzeug.utils import secure_filename

filename = secure_filename(file.filename)
file.save(os.path.join(app.config["UPLOAD_FOLDER"], filename))
