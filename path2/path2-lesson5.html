Lesson 5: Flask Templates & Rendering HTML

1. What are Templates?

Templates are HTML files used to separate design (frontend) from Python code (backend).

Flask uses Jinja2 template engine by default.

2. Flask Template Folder

project_folder/
│── app.py
│── templates/
│    index.html
│    about.html
Flask automatically looks for templates inside templates/ folder.

3. Rendering HTML

from flask import Flask, render_template
app = Flask(__name__)

@app.route("/")
def home():
  return render_template("index.html")
Flask renders index.html from templates folder.

4. Passing Data to Template

@app.route("/user/<name>")
def user(name):
  return render_template("user.html", username=name)
username=name passes data to HTML template.

5. Using Variables in Template

<!-- user.html -->
<h1>Hello, {{ username }}!</h1>
{{ username }} is replaced by the value passed from Flask.

6. Control Structures in Templates

<ul>
{% for item in items %}
  <li>{{ item }}</li>
{% endfor %}
</ul>
You can use loops and conditionals inside templates using Jinja2 syntax.

Practice Tasks

  1. Create index.html template and render from Flask
  2. Pass a variable and display using {{ }}
  3. Create a list in Flask and display using {% for %} loop
  4. Try using {% if %} conditionals
