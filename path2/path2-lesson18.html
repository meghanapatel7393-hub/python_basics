Lesson 18: Flask REST API with Database Integration

1. Why Database Integration?

Storing API data in a database ensures persistence across server restarts and allows scalability.

2. Setting Up SQLAlchemy

from flask import Flask, request, jsonify
from flask_sqlalchemy import SQLAlchemy

app = Flask(__name__)
app.config["SQLALCHEMY_DATABASE_URI"] = "sqlite:///todos.db"
app.config["SQLALCHEMY_TRACK_MODIFICATIONS"] = False
db = SQLAlchemy(app)
SQLAlchemy is an ORM that maps Python objects to database tables.

3. Creating a Database Model

class Todo(db.Model):
  id = db.Column(db.Integer, primary_key=True)
  task = db.Column(db.String(200), nullable=False)

# Create database tables
with app.app_context():
  db.create_all()

4. GET Request – Fetch Todos

@app.route("/todos", methods=["GET"])
def get_todos():
  todos = Todo.query.all()
  result = [{"id": t.id, "task": t.task} for t in todos]
  return jsonify(result)

5. POST Request – Add Todo

@app.route("/todos", methods=["POST"])
def add_todo():
  data = request.get_json()
  new_todo = Todo(task=data["task"])
  db.session.add(new_todo)
  db.session.commit()
  return jsonify({"id": new_todo.id, "task": new_todo.task}), 201

6. PUT Request – Update Todo

@app.route("/todos/<int:id>", methods=["PUT"])
def update_todo(id):
  todo = Todo.query.get(id)
  if not todo:
    return jsonify({"error": "Not found"}), 404
  data = request.get_json()
  todo.task = data["task"]
  db.session.commit()
  return jsonify({"id": todo.id, "task": todo.task})

7. DELETE Request – Remove Todo

@app.route("/todos/<int:id>", methods=["DELETE"])
def delete_todo(id):
  todo = Todo.query.get(id)
  if not todo:
    return jsonify({"error": "Not found"}), 404
  db.session.delete(todo)
  db.session.commit()
  return jsonify({"message": "Deleted"})

8. Notes & Best Practices

  • Always commit session changes with db.session.commit()
  • Use app.app_context() when creating tables outside routes
  • Validate incoming JSON data for required fields
  • Consider using Marshmallow or Pydantic for schema validation
