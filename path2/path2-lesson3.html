Lesson 3: Flask Installation & First App

1. What is Flask?

Flask is a lightweight Python web framework that helps you build web apps and APIs quickly.

Flask is called a "micro-framework" because it does not include ORM or built-in authentication.

2. Install Flask

# Create virtual environment
python -m venv venv

# Activate environment
# Windows: venv\Scripts\activate
# Linux/Mac: source venv/bin/activate

# Install Flask
pip install Flask

3. First Flask App

from flask import Flask

app = Flask(__name__)

@app.route("/")
def home():
  return "Hello, Flask!"

if __name__ == "__main__":
  app.run(debug=True)
@app.route("/") defines a URL endpoint.
debug=True enables auto-reload and error messages.

4. Run the App

python app.py
Open browser → http://127.0.0.1:5000/ → You will see "Hello, Flask!"

5. Flask App Structure (Simple)

project_folder/
│── app.py
│── venv/
│── requirements.txt
Small projects can have just one file. Larger projects use folders & templates.
