Phase 1 – Web Fundamentals

Lesson 1: How the Web Works

1. What is a Website?

A website is a collection of web pages served to users through the internet.

2. Client & Server

  • Client: Browser (Chrome, Edge)
  • Server: Computer that runs backend code
Backend developer works on the server.

3. Request & Response

Browser → Request → Server Server → Response → Browser

4. HTTP Methods

  • GET – Read data
  • POST – Send data
  • PUT – Update data
  • DELETE – Remove data

5. Backend Role

  • Handle requests
  • Process logic
  • Connect to database
  • Send response

6. Where Flask/Django/FastAPI Fit

  • Flask – Lightweight backend
  • Django – Full framework
  • FastAPI – High-speed APIs
