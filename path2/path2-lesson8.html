Lesson 8: Flask REST API Basics – JSON Response

1. What is REST API?

REST API allows clients (like browsers or mobile apps) to communicate with a server using HTTP requests and receive structured data (JSON).

REST APIs are widely used in modern web applications.

2. JSON Response in Flask

from flask import Flask, jsonify
app = Flask(__name__)

@app.route("/api")
def api():
  data = { "name": "Bhavesh", "role": "Backend Developer" }
  return jsonify(data)
Visiting /api returns JSON: {"name": "Bhavesh", "role": "Backend Developer"}

3. REST API with Dynamic Parameters

@app.route("/api/user/<username>")
def get_user(username):
  data = { "user": username, "status": "active" }
  return jsonify(data)
Visit /api/user/Bhavesh → {"user": "Bhavesh", "status": "active"}

4. HTTP Methods with REST API

from flask import request

@app.route("/api/data", methods=["POST"])
def post_data():
  data = request.get_json()
  return jsonify({ "received": data })
You can send JSON from client and receive it in Flask using request.get_json()

5. Mini Example – Todo API

todos = [
  {"id": 1, "task": "Learn Flask"},
  {"id": 2, "task": "Build API"}
]

@app.route("/api/todos")
def get_todos():
  return jsonify(todos)
Visit /api/todos → returns list of tasks in JSON format
