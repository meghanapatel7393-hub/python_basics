Lesson 14: Regular Expressions (Regex)

1. What is Regex?

Regex (Regular Expression) is a sequence of characters that forms a search pattern. Used for text searching, validation, and manipulation.

Interview line:
“Regex is widely asked for string validation and data extraction.”

2. Importing Regex Module

import re

3. Basic Regex Functions

Function Purpose
re.match() Match pattern at the beginning of string
re.search() Search pattern anywhere in string
re.findall() Return all occurrences of pattern
re.split() Split string by pattern
re.sub() Replace pattern with another string

4. Basic Patterns

  • abc – Matches literal "abc"
  • ^ – Start of string
  • $ – End of string
  • . – Any character except newline
  • * – 0 or more repetitions
  • + – 1 or more repetitions
  • ? – 0 or 1 repetition
  • [a-z] – Any lowercase letter
  • \d – Digit (0-9)
  • \w – Word character (a-z, A-Z, 0-9, _)

5. Examples

import re
pattern = r"\d+" # Matches 1 or more digits
text = "My number is 12345"
result = re.findall(pattern, text)
print(result) # Output: ['12345']
# Email validation
pattern = r"^[\w\.-]+@[\w\.-]+\.\w+$"
text = "test@example.com"
if re.match(pattern, text):
  print("Valid email")
else:
  print("Invalid email")

6. Compiling Patterns

pattern = re.compile(r"\d+")
result = pattern.findall("There are 12 cats and 5 dogs")
print(result) # ['12', '5']
Compiling improves performance if pattern is used multiple times.

7. Regex Tips

  • Use raw string r"pattern" to avoid escape issues
  • Always test your regex patterns
  • Use re.VERBOSE for complex patterns
